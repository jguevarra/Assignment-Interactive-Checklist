Goals

The Monkey's With Typewriters team goal was to find the solution to making student life easier and more organized by doing exactly that for them. Given the opportunity to take on, develop, and design this project, we solved the problem of the large number of clicks to find the due date of an assignment on UVA's Collab website. The assignment to a respective class will be found under the tab for that class, causing the large amount of clicks to find the due dates for all assignments.

Feature #1: View All Assignments on One Page

Once users enroll in all their appropriate classes for the semester, they will be able to view all the assignments from their enrolled classes in one page under their "To Do" list at the bottom of the page. The complete list of their assignments was programmed to be sorted chronologically by due date to help users prioritize which assignments to get done first and which ones are upcoming.

Feature #2: To Do List

Alongside view all their assignments, when the user click on the assignment, the assignment will be checked off and crossed off with a strike-through line, indicating that the user completed the assignment. Furthermore, the user can click on the "x" mark on the right side of the line to permanently delete the assignment from the assignment list.

Feature #3: Post New Assignments

Lastly, the user can not only just view assignments, they can also post new assignments to the classes that they are enrolled in. There is a form that will allow them to pick the class they are posting for, and input the name, due date, due time, and the description of the assignment. These details will be displayed when the user clicks on the link of the assignment post.
